# 🌟 Overview

TalentOS AI is an **Enterprise AI Hiring Intelligence Platform** designed to modernize recruitment using **Azure AI Foundry**, **Large Language Models**, **AI Agents**, and **Retrieval-Augmented Generation (RAG)**.

Instead of functioning as a simple chatbot, TalentOS AI assists recruiters throughout the hiring lifecycle by intelligently analyzing resumes, discovering top candidates, conducting AI-powered interviews, and generating structured hiring reports.

The project demonstrates how modern enterprise AI applications are architected by combining multiple Azure AI services into a single production-style platform.

---

# 🎯 Project Vision

Recruiters spend countless hours:

* Reading resumes
* Comparing candidates
* Creating interview questions
* Evaluating applicants
* Writing hiring reports

TalentOS AI automates these repetitive tasks while helping recruiters make faster, more informed, and more consistent hiring decisions.

# ✨ Core Features

## 📄 AI Resume Intelligence

* Upload resumes
* AI-powered resume parsing
* Structured candidate profiles
* Skills extraction
* Experience extraction
* Education analysis
* Professional summaries

---

## 🔍 Semantic Candidate Search (RAG)

Move beyond traditional keyword search.

TalentOS AI enables recruiters to search naturally:

> "Find Python developers with Azure AI experience."

> "Show candidates experienced in Machine Learning and FastAPI."

Powered by:

* Azure AI Search
* Embeddings
* Vector Search
* Hybrid Search
* Retrieval-Augmented Generation

---

## 🤖 AI Recruiter Copilot

An intelligent assistant capable of:

* Ranking candidates
* Comparing applicants
* Highlighting strengths
* Identifying skill gaps
* Recommending interview focus areas
* Explaining hiring decisions

---

## 🎤 AI Interview Assistant

Automatically:

* Generates technical questions
* Creates behavioral questions
* Produces follow-up questions
* Evaluates answers
* Scores candidates
* Generates interview summaries

---

## 🧠 Azure AI Foundry Recruiter Agent

One of the platform's flagship capabilities.

Rather than acting as a chatbot, the Recruiter Agent orchestrates multiple business workflows.

The agent can:

* 📄 Read uploaded resumes
* 🔍 Search the candidate database
* 📊 Compare applicants
* ❓ Generate interview questions
* 📝 Produce recruiter reports

This mirrors how enterprise organizations build AI agents to coordinate existing business capabilities instead of embedding all logic in a single prompt.

---

## 📊 Prompt Evaluation

TalentOS AI leverages Azure AI Foundry Evaluations to:

* Compare prompt versions
* Measure AI quality
* Improve reliability
* Validate outputs
